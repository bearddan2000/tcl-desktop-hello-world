# tcl-desktop-hello-world

## Description
This is a POC project to demonstrate tcl gui builder.

Supports X11 display forwarding from docker container.

## Tech stack
- tcl

## Docker stack
- ubuntu

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
